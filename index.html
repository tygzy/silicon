CSS without the requirement for classes. there are classes in silicon, but you could use silicon purely with no classes, just tags such as <div>. fonts are also included in the css file, so you don't need to link any.

silicon also has a light and dark mode that switches with the clients operating systems preference

a card with borders, could be used to separate content from something else.
make this with <div class='card'>

a container element with an opposing background.
make this with <div class='background'>

<div class='flex'> makes any element a flexbox with vertical and horizontal center alignment.

<div class='wide'> makes any element 100% width relative to it's parent.

<div class='left / center / right'> aligns any text to the left / center / right.

<div class='no-resize / all-resize / ho-resize / ve-resize'> allows or disallows a textarea to be resized in specified directions.
